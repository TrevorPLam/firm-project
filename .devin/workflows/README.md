# Devin Workflows

Custom workflow definitions for AI coding assistants (Cascade, Devin, etc.).

## Purpose

Contains workflow configuration files that define automated processes and task execution patterns for AI coding assistants (Cascade, Devin, etc.).

## Structure

- `todo.md` - Workflow definition for executing TODO.md tasks with research, best practices, QA, and commit

## Usage

Workflows are invoked via slash commands (e.g., `/todo`) within your AI assistant interface to execute predefined automation sequences.

## Notes

Workflows follow a YAML frontmatter format with description and step-by-step execution instructions.
