# Cursor Hooks

Cursor IDE hook configuration and state management.

## Purpose

Contains configuration files and state data for Cursor IDE's hook system, which enables automated behaviors and custom integrations within the Cursor development environment.

## Structure

- `state/` - Hook state and runtime data

## Files

- `state/README.md` - Documentation for the state subdirectory
- `state/continual-learning.json` - Cursor's continual learning feature state tracking

## Notes

This directory is managed by Cursor IDE and should not be manually edited unless you understand the hook system's internal state management.
